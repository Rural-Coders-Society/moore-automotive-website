# Moore Automotive Website

Single-page website for **Moore Automotive** with a call-first layout.

## Includes
- Hero section with prominent call-to-action
- About section
- Services section
- Testimonials section
- No email/contact form (phone-only contact)

## Run locally
```bash
python3 -m http.server 8123
```
Then open `http://127.0.0.1:8123/index.html`.
